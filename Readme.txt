These are the supplementary materials for "Why the South Fought: The Myth of Southern Poverty before the Civil War" by Joseph Francis. The Master.R file will run the following scripts:

Download_IPUMS.R
This downloads the necessary data from IPUMS and saved them in the Downloads and Data folders. You will need an API key from https://account.ipums.org/api_keys.

Replication_v2.R
This processes the data and produces Replication.txt and Maps.gpkg in the Results folder.
